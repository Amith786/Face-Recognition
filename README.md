# Face Recognition using OpenCV

A real-time face recognition system developed using Python and OpenCV. The project detects and recognizes faces from a live webcam feed using the Haar Cascade Classifier and the Local Binary Pattern Histogram (LBPH) Face Recognizer.

## Features

- Real-time face detection
- Face recognition using LBPH
- Webcam-based recognition
- Dataset generation
- Model training
- Confidence score display

## Technologies Used

- Python
- OpenCV
- NumPy
- Haar Cascade
- LBPH Face Recognizer

## Applications

- Smart Attendance System
- Access Control
- Identity Verification
- Security and Surveillance
